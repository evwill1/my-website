Software for astrophotography

Aquisition

Aquiring images requires some form of software known as image capture or aquisition software. Some software is free while others can cost hundreds of dollars. Depending on the needs of the photographer, you may be able to just use free capture software.

Here is a list of aquisition software

  • Astro Photography Tool (APT)
  • NINA
  • Sequence Generator Pro
  • SharpCap
  • ZWO ASI Studio (using ASIAir, Pro, or Plus)
  • Backyard EOS (for Canon Cameras)
  • Backyard Nikon (for Nikon Cameras)

Tracking

For a more automated process, guiding software is necessary for tracking. Tracking software is used to control a motorized telescope mount to remain fixed on the target object as the camera takes many photos with as little errors as possible so images come out nice and clean.

The most popular tracking software in the field is PHD2.

Stacking

After a night of imaging a photographer is left with a tens, possibly hundreds of images in a .RAW or .FITS file type. These are uncompressed and unprocessed image files that contain the valuable image data even if the do not look like it. This is where stacking software comes in for the win. These softwares use intelligent algorithms to detect and align stars within a photo and literally stack them together. This is useful for two reasons-1. it removes unwanted artifacts such as satellite and plane trails. and 2. it enhances the details of the target object by increasing the signal in the photo.

Popular stacking software includes:

  • DSS: deep sky stacker
  • WBPP: Weighted Batch Pre-Processing (This is a script contained within Pixinsight processing software)

Processing

Processing is the last step in the processing of creating a astrophoto. This is the point where the photographer gets to express their artistic side while maintaining some scientific dedication. The processing phase usually includes cropping, rotating, background extraction, gradient removal, histogram and curves transformation, and color balancing.

The most popular programs for processing are:
  • Pixinsight
  • Adobe Photoshop
  • Siril
  • Gimp
